## Getting Started

### Prerequisites

This project requires the following dependencies:

- **Programming Language:** Java
- **Package Manager:** Maven
- **Container Runtime:** Docker

### Installation

Build repo from the source and intsall dependencies:

1. **Clone the repository:**

    ```sh
    ❯ git clone ../repo
    ```

2. **Navigate to the project directory:**

    ```sh
    ❯ cd repo
    ```

3. **Install the dependencies:**

<!-- SHIELDS BADGE CURRENTLY DISABLED -->
    <!-- [![docker][docker-shield]][docker-link] -->
    <!-- REFERENCE LINKS -->
    <!-- [docker-shield]: https://img.shields.io/badge/Docker-2CA5E0.svg?style={badge_style}&logo=docker&logoColor=white -->
    <!-- [docker-link]: https://www.docker.com/ -->

    **Using [docker](https://www.docker.com/):**

    ```sh
    ❯ docker build -t app/repo .
    ```
<!-- SHIELDS BADGE CURRENTLY DISABLED -->
    <!-- [![maven][maven-shield]][maven-link] -->
    <!-- REFERENCE LINKS -->
    <!-- [maven-shield]: https://img.shields.io/badge/Maven-C71A36.svg?style={badge_style}&logo=apache-maven&logoColor=white -->
    <!-- [maven-link]: https://maven.apache.org/ -->

    **Using [maven](https://maven.apache.org/):**

    ```sh
    ❯ mvn install
    ```

### Usage

Run the project with:

**Using [docker](https://www.docker.com/):**
```sh
docker run -it {image_name}
```
**Using [maven](https://maven.apache.org/):**
```sh
mvn exec:java
```

### Testing

Repo uses the {__test_framework__} test framework. Run the test suite with:

**Using [maven](https://maven.apache.org/):**
```sh
mvn test
```
